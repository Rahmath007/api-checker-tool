# API Checker Tool

A simple Python script for checking API health by sending GET requests and printing HTTP status codes.  
Useful for Technical Support and Application Support roles.

## Features
- Asks the user to enter an API URL
- Sends a GET request using `requests`
- Prints the HTTP status code (200, 404, 500, etc.)
- Handles timeouts and connection errors
- Clean command-line output

## How to Run

```bash
python api_checker.py
